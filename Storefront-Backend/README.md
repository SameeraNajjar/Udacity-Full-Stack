# MyStore - Angular Ecommerce Application

MyStore is a full-featured single-page ecommerce application built with Angular. It allows users to browse products, view product details, add items to cart, and complete the checkout process.

## Features

- **Product Catalog**: Browse available products with images, prices, and descriptions
- **Product Details**: View detailed information about each product
- **Shopping Cart**: Add/remove products and manage quantities
- **Checkout Process**: Secure checkout form with validation
- **Order Confirmation**: Success page after order placement
- **Responsive Design**: Works on all device sizes

## Installation & Launch

1. Install dependencies:
```bash
npm install
```

2. Launch the application:
```bash
ng serve
```

3. Open your browser and navigate to:
```
http://localhost:4200
```

## Technologies Used
- Angular 17+
- TypeScript
- RxJS
- Bootstrap for styling
- ngx-toastr for notifications

## Notes
- Cart component has been fixed and improved to handle quantity updates, removal of items, and total calculation properly.
